# 브랜드 기반 (Brand Foundation)

> 로고·이름보다 **먼저** 읽는다.  
> 브랜드 창출의 핵심은 속성 선택이 아니라, 그 선택이 가리키는 **차별·감정·가치 공유**다.  
> 수십 년에 걸쳐 쌓일 **실체감**을 위해, 모든 접점이 같은 신을 가리키게 한다.

|------|----------------------|------|
| 벨 | 성장의 챕터 | 후보에서 제외 |

| 글러브 | 장비 | 후보에서 제외 |

## 톤 — 절제된 힘 (잠금에 가깝게)

나이키·아디다스를 기준으로 보면:

- **담담만** 있는 브랜드가 아니다. 힘·자신감·승부가 있다.
- 다만 **소리 지르지 않는다.** 그래픽은 깔끔하고, 말은 짧다.
- 우리: **파워풀 + 절제.** 오바(핏·잔혹·잠재력 폭발·네온)는 거절.

| | 한다 | 안 한다 |
|--|------|---------|
| 힘 | grit, 올라간다, 증명 | 피·폭력 미학, 협박 라벨 |
| 절제 | 짧은 슬로건, 최소 도형 마크 | 설명형 일러스트, 과한 서사 겹침 |
| 일상 UI | 숨 고르는 담담함 OK | 브랜드 얼굴까지 전부 조용할 필요 없음 |

`Show your grit.` / 링에 올라 증명 = 이 톤과 맞음.  
사람+핏방울 글러브 = 힘은 있는데 **절제 밖으로 나가기 쉬움** → 마크보단 특수 연출용.

## 브랜딩이란 (우리 정의)

1. **차별** — 하나의 제품을 타사와 구분하려고 네임·로고·심벌·패키지·기타 속성을 **신중히** 고른다.
2. **관계** — 소비자의 심중을 관통해 감정을 교류하고, 관계를 시작한다.
3. **가치 공유** — 브랜드와 소비자가 같은 가치를 갖는다.
4. **경험** — 앱 디자인, 패키지, 문구·화법 등 **브랜드를 경험하게 하는 모든 것**이 그 가치를 품는다.
5. **시간** — 하루 만에 완성되지 않는다. **수십 년 쌓여** 단단한 실체감이 된다. 그래서 처음부터 방향을 흔들리지 않게 둔다.

로고는 브랜딩의 **일부**다. 브랜딩 전체가 아니다.

## 0-1. 무엇을 차별하는가

타사가 팔고 우리가 **안** 파는 것 / 우리가 팔고 타사가 **약하게** 파는 것.

| | 타사(전형) | 우리 |
|--|-----------|------|
| 정체 | 피트니스 트래커 / 복싱 게임 | **복싱 인생의 기록·증명** |
| 사용자 | 레벨 올리는 플레이어 | **이미 복서인 주인공** |
| 감정 | 환호·퀘스트·티어 | **담담함 · 오늘도 벨** |
| 자랑 | SNS 좋아요 | **몇 년 뒤 스스로 돌아보는 흔적** |
| 표면 | 대시보드·스탯 | **장면·스틸컷** |

한 줄 차별:

> 우리는 복싱을 **수치로 이기게** 하지 않는다.  
> 복싱을 **살아온 시간으로 남기게** 한다.

## 0-2. 공유 가치 = 우리만의 문구 (크리드)

브랜드 ↔ 소비자가 악수하는 지점. **뼈대 문장 하나.**  
부드러은 V1–V4(“멈추지 않는다”류)는 **보류**.  
원하는 결: 네이비씰 · UDT · 《인간 실격》처럼 — **빗나가 있고, 심장을 찌른다.** 복싱 앱만의 것.

레퍼런스(훔치지 않음, 결만 빌림):

| 결 | 예시 | 우리가 가져올 것 |
|----|------|------------------|
| 씰 | 고통은 나약함이 몸에서 빠져나가는 것 / 편한 날은 어제뿐 | 고통·어제·변명의 거절 |
| UDT | 불가능은 없다 | 단호함 — 단, 아디다스 카피 복제 금지 |
| 인간 실격 | 부끄럼 많은 생 / 겁쟁이는 행복도 두렵다 | 수치·두려움 — 그래도 링에 선다 |

### 후보 (우리 문장 — 미확정, 하나만 고른다)

**A. 씰 결 — 어제·고통**

| ID | 문구 |
|----|------|
| A1 | **편한 라운드는 어제 것이다.** |
| A2 | **아프면, 나약한 게 빠져나가는 거다.** |
| A3 | **쉬운 벨은 없다.** |

**B. 단호 결 — 불가능·거절**

| ID | 문구 |
|----|------|
| B1 | **링 위엔 불가능이 서지 못한다.** |
| B2 | **안 된다고 말한 적은, 글러브를 끼기 전뿐이다.** |
| B3 | **멈춘 적이 없다. 쉬었을 뿐이다.** |

**C. 인간실격 결 — 수치·겁·그래도**

| ID | 문구 |
|----|------|
| C1 | **겁이 나서 왔다. 그래서 복서다.** |
| C2 | **부끄러운 라운드도, 라운드다.** |
| C3 | **행복이 두려운 날에도, 벨은 울린다.** |
| C4 | **약한 생을 보냈어도, 오늘 벨은 내 것이다.** |

**D. 우리 영화 결 — 록키·담담 + 엇지**

| ID | 문구 |
|----|------|
| D1 | **오늘도 벨은 울렸다. 그게 전부다.** |
| D2 | **이긴 날이 아니라, 온 날을 센다.** |
| D3 | **관객은 없다. 벨만 있다.** |
| D4 | **네 복싱은, 네가 미안해하는 그 시간에도 켜져 있다.** |

### 권장 (초안)

| 역할 | 문구 | 이유 |
|------|------|------|
| **뼈대 (크리드)** | **C3 — 행복이 두려운 날에도, 벨은 울린다.** | 인간실격의 빗김 + 우리 벨. 타 피트니스 앱에 없음. |
| 보조 (일상 완료) | D1 — 오늘도 벨은 울렸다. 그게 전부다. | 담담한 일상 화법 |
| 보조 (씰 톤 필요 시) | A1 — 편한 라운드는 어제 것이다. | 훈련 직전·하드 세션 |

뼈대는 **하나**. 보조는 화법에서만. 로고에 문장을 다 넣지 않는다.

**쓰지 않는 것:** 씰/UDT 문장 직역 도용, “불가능은 없다” 그대로, “퀘스트 클리어”, 매 라운드 환호.

## 0-2b. 복싱하면 떠오르는 것 (사용자 정의 · 감정 기둥)

사물(글러브·벨)이 아니라, **느낌 네 가지.** 브랜드가 팔 기분.

## 0-4. 브랜드 경험 접점 (모든 것이 브랜드다)

속성은 “예쁜 로고”가 아니라, **가치·감정이 닿는 면**이다.

| 접점 | 역할 | 지금 상태 |
|------|------|-----------|
| 네임 | 부르는 소리 | 미잠금 (`ROUND ON` 임시) |
| 로고·심벌 | 한눈에 보이는 얼굴 | 브리프 단계 |
| 앱 UI | 매일 입는 옷 | 철학 문서 있음 · 색은 흔들림 |
| 문구·화법 | 말투 | 철학 체크리스트 있음 |
| 공유·포스터 카드 | 밖에서의 얼굴 | 부분 구현 |
| 온보딩 | 첫 악수 | 있음 · 브랜드 정렬 필요 |
| 패키지·실물 (후) | 손끝 실체 | 없음 |
| 마케팅 | 이야기 | 이름·마크 이후 |

**원칙:** 접점마다 다른 브랜드를 말하지 않는다.  
수십 년 뒤 “이 앱 특유의 공기”가 남으려면, 오늘 고른 가치(V)가 모든 접점에 같은 방향으로 스며야 한다.

## 0-5. 실체감을 위한 시간 규칙

- 유행 속성(그해의 힙한 UI·로고)으로 기반을 갈아엎지 않는다.
- 속성(색·아이콘·카피)은 **V(공유 가치)에 복종**한다. 반대로 가지 않는다.
- 작은 일관성 × 긴 시간 = 실체감. 화려한 한 방 × 자주 바꾸기 = 안개.
