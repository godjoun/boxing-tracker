## ① 사업계획서 1페이지

### 서비스명 (가칭)
**복싱 트래커 — 홈복싱·도장을 잇는 파이터 OS**

### 한 줄 요약
홈복싱·입문자가 **훈련을 기록·증명**하고, **커리큘럼으로 배우며**, **도장·코치와 연결**되는 모바일 앱.

### 문제
- 홈복싱은 혼자 하면 동기부족·기록 없음·도장 가기 막막함
- 복싱장은 신규 유입·체험 전환 비용이 큼
- 범용 피트니스 앱(플랜핏 등)은 복싱 정체성·도장 연결이 약함

### 솔루션
| 기능 | 가치 |
|------|------|
| 라운드 타이머 + EXP/레벨 | 훈련 습관·성장 가시화 |
| 인증 카드 | SNS 공유·자랑(바이럴) |
| 4주 홈복싱 커리큘럼 | 운동 안 하는 날도 「오늘의 레슨」 |
| 도장깨기·스파링 | 오프라인 연결(중개 수익) |

### 타겟 고객
20~35세, 홈복싱·복싱 입문, 도장 관심은 있으나 부담 있는층 (1차)

### 수익 모델 (Phase 1 — AI 구독 없음)

→ 상세 구조: **「①-2 무료 입문 vs 유료 코스」** 참고

1. **도장 리드 중개** — 체험 문의 연결 시 건당/월 정액
2. **코치 프로그램** — 4주 홈복싱 등 유료 패키지, 수수료 15~30%
3. **콘텐츠** — 무료 샘플(입문 4주) + 유료 강의 패키지

### 경쟁·차별
- 플랜핏: AI 헬스 구독 → 우리는 **복싱 특화 + 정체성 + 오프라인**
- 파이트캠프: 장비+영상 구독(해외) → 우리는 **가볍게 시작 + 한국 도장 연결**

### 현재 단계
- MVP 배포 (Vercel), 타이머·기록·성장·인증 카드·커리큘럼 동작
- 로컬 우선, 닉네임 Supabase 선택 연동
- **다음:** 오늘의 레슨·주간 리포트·도장 리드 파일럿

## ①-2 무료 입문 vs 유료 코스 (핵심 구조)

> **앱은 계속 무료.** 더 배우고 싶을 때만 코스를 산다.  
> 월 AI 구독이 아니라, **「프로그램(4주 패키지)」판매 + 도장 연결**이 Phase 1 수익 모델이다.

### 쉬운 비유

| | 헬스장 비유 | 복싱 트래커 |
|---|------------|-------------|
| **무료** | 입장 + 기구 | 타이머, 기록, EXP, 인증 카드 |
| **무료** | 입문 프로그램 표 | **홈복싱 입문 4주** (12세션) |
| **유료** | PT·전문 클래스 | 코치/전문 **유료 코스** (중급 이상) |
| **중개** | 지점 연결 | **도장 체험 리드** |

### 사용자 여정

```
1. 앱 설치 (무료)
2. 타이머 훈련 + 입문 4주 따라하기 (무료)
3. 더 배우고 싶으면 → 「스파링 준비 4주」 등 유료 코스 구매
4. 실전·도장까지 가고 싶으면 → 도장·코치 연결 (앱 소개)
```

### 앱 vs 코스 역할

| 담당 | 내용 |
|------|------|
| **앱 (플랫폼)** | 타이머, 기록, 레벨, 진행률, 결제·온보딩, 도장 연결 |
| **코치/제작자** | 프로그램 기획, 영상·드릴, (선택) 피드백 |
| **수익** | 코스 매출 배분 (파일럿: 코치 70% / 플랫폼 30%) |

중급·고급은 **앱 기능을 무한히 키우는 게 아니라**, **유료 코스를 추가**하는 방식으로 확장한다.

### Phase 1에서 하지 않는 것

- 월정액 AI 코칭 구독 (플랜핏 경로)
- 전 레벨을 무료 앱 하나에 전부 넣기
- 하드웨어·펀치 트래커

### 지금 vs 나중

| 시기 | 무료 | 유료 |
|------|------|------|
| **지금 (베타)** | 입문 4주 + 타이머 | 코치 파일럿 1건 준비 |
| **6개월** | 동일 (품질·영상 보강) | 유료 코스 1~2개 + 도장 리드 |

## ② 도장 제휴 제안 메시지

**제목:** 홈복싱 앱 사용자 체험 문의 연결 제안

안녕하세요, **[도장명]** 관장님(코치님).

저희는 **홈복싱 입문자**를 위한 훈련 기록 앱 「복싱 트래커」를 운영하고 있습니다.  
앱에서 가까운 복싱장을 찾는 「도장깨기」 기능을 준비 중이며, **체험·입문 상담을 원하는 분을 도장으로 연결**해 드리고 싶습니다.

**도장에 드리는 점**
- 홈에서 혼자 하던 입문자 → **실제 도장 체험**으로 연결
- 별도 광고비 없이 **관심 있는 지역 유저**만 전달
- (협의) 연결 성공 시 **건당 ○○원** 또는 **월 제휴 ○○원**

**저희가 부탁드리는 점**
- 체험 1회 프로그램(또는 입문 클래스) 안내
- 연결된 분에게 **첫 방문 시 환영** (앱에서 소개한 회원임을 인지)

- **수익 배분:** 매출의 **70% 코치 / 30% 플랫폼** (협의 가능)

**1차 파일럿**
- 기간 4주, 정원 10~20명, 가격 **5~10만 원** 테스트
- 성과 좋으면 마켓에 정식 등록

15분 통화로 궁금하신 점 나눠 주시면 감사하겠습니다.
