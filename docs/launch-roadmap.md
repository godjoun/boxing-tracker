## 전체 타임라인 (16주)

```
주 1~2   │ 제품 A — 영상 1~2개 + URL 반영
주 3~4   │ 제품 A — 홈「영상만 보기」+ 베타 5명 시작
주 5~8   │ 베타 운영 + 영상 4~6개 + 피드백 반영
주 9~12  │ 소프트 출시 + 도장/코치 파일럿 1건
주 13~16 │ 숫자 정리 + 예비창업패키지 / 엔젤 준비
```

## 투자 · 출시 · 베타 순서 (정리)

```
지금 ──► Phase 0~1 (영상 + 베타 5명)        ← 출시 전
      ──► Phase 2 (베타 운영 + 숫자)        ← 클로즈드 베타
      ──► Phase 3 (소프트 출시 + 파일럿)    ← 「출시」에 가까움
      ──► Phase 4 (정부 지원 / 엔젤)        ← 숫자 붙여서 신청
      ──► (훨씬 뒤) VC 시드                 ← MAU·매출 후
```

| 돈 | 언제 |
|----|------|
| 예비창업패키지 | Phase 1~2 중 신청 가능 (출시 전 OK) |
| 소액 엔젤 | Phase 2~3, 베타 숫자 있을 때 |
| VC | Phase 4 이후도 이른 편, 매출·성장 후 |
