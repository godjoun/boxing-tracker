### 교류에서 미리 알릴 리스크 (계속 체크)

- 예시(시드) 카드와 진짜 일정이 섞여 헷갈릴 수 있음 → 진짜 생기면 예시 숨기기 검토

- 신청해도 주최자 **알림·채팅** 없음 (흐름 4번)
- env 바꾼 뒤엔 `npm run dev` **재시작** 해야 서버 연결됨
