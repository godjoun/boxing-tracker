# 검증 후 자동 과금 게이트

현재 상태: **잠금 — 구현하지 않음**

아래 증거가 없으면 관장 인증, `featured_until`, 요금제, 결제 SDK·웹훅을
추가하지 않는다.

- [ ] 14일 파일럿 종료
- [ ] 추천 상세 진입 2배 / 체육관당 주 1 유효 문의 / 3곳 유료 지속 의사 중 하나 충족
- [ ] 실제 지불액과 환불·중단 사례 기록
- [ ] 운영자가 성공 판정 승인

게이트 통과 후 순서:

1. Supabase Auth 기반 관장 계정과 입점관 소유권 인증
2. `featured_until`과 서버 시간 기준 추천 노출 권한
3. 단일 요금제·기간·환불 정책
4. 결제 주문과 권한을 분리한 서버 스키마
5. 서명 검증·중복 처리 방지 결제 웹훅
6. 실패·취소·환불 시 추천 권한 회수
7. 운영 장부와 감사 로그

기기 actor UUID는 인증 수단이 아니므로 유료 권한에 사용하지 않는다.

스타일 프리미엄은 별도 수요 검증 대상이다. 기존 텍스트 드릴과 타이머는
무료로 유지하고, 새 영상·심화 6~8주 프로그램만 후보로 둔다.
