# 도장 화면 구조 (참고 분석 → 우리 반영)

> UI 복제 금지. **흐름·자리만** 빌린다.  
> 제품: `docs/dojo.md` · 수익: `docs/monetization.md`

## 참고 앱에서 가져온 것

| 앱 | 빌린 구조 | 안 가져온 것 |
|----|-----------|--------------|
| **당근** 비즈프로필 | 찾기 → **가게 홈**(커버·소개·가격) → 문의 | 단골·쿠폰·후기 피드 |
| **플랩** | 장소/매치 카드 → **상세** → 신청(행동 1개) | 소셜 매치·레벨·구장 예약 풀스택 |
| **Boxing Network** | 짐/이벤트 **허브** + 문의·매칭은 다음 행동 | 팀 가입·bout sheet·코치 워크플로 |
| **더코너류** | (나중) **관에 붙는** 소식·멤버 | 지금 전역 커뮤니티 피드 |

## 우리 화면 3장 (잠금)

```
1. 찾기 (목록)     ← 지역·GPS · 추천 슬롯 · 카드
2. 관 한 장면      ← 사진 · 이름 · 소개 · 가격 · 문의 CTA
3. (나중) 관 소식  ← 커뮤니티는 상세 안 슬롯에만 자리
```

문의 모달은 2번 위(또는 목록 빠른 버튼)에서 연다.  
연락처는 공개하지 않는다.
